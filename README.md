# workbook express+mongodb+vue

这是一个完整的工作日志系统，包括后台接口的提供、后台管理界面、用户操作界面。

**开启api**

1. cd server

2. npm i

3. npm start

**后台管理界面**

1. cd manager

2. npm i

3. npm start

4. 在浏览器中键入:http://localhost:9001 进入后台管理界面

**用户界面**

1. cd client

2. npm i

3. npm start

4. 在浏览器中键入:http://localhost:9001 进入用户界面
